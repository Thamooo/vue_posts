# Тестовое задание

## Установка

### Пакеты

```
npm install
```

### Запуск

#### Клиентская часть:

```
npm run serve
```
По умолчанию проект будет доступен по данному адресу
```
http://localhost:8080/
```

#### Серверная часть:

```
json-server --watch db.json
```

Обратите внимание, что обе команды должны быть запущенны одновременно в разных консолях или вкладках.
